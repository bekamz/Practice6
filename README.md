# Practice 6 – Python File Handling and Built-in Functions

This repository contains examples for Practice 6.

## Topics Covered

### 📂 File Handling
- File modes: r, w, a, x
- Reading files: read(), readline(), readlines()
- Writing and appending files
- Copying and deleting files (shutil)
- Working with pathlib

### 📁 Directory Management
- Creating directories
- Listing files and folders
- Moving and copying files
- Searching files by extension

### 🔢 Built-in Functions
- len(), sum(), min(), max()
- map(), filter(), reduce()
- enumerate(), zip()
- sorted()
- Type checking and conversion

All examples are organized into separate folders:
- file_handling
- directory_management
- builtin_functions
